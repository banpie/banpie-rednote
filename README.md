# Banpie RedNote

半撇风格小红书图文助手 - Obsidian 插件

## 功能特性

- 🎨 **多模板支持**
  - 默认模板：简洁风格，适合常规笔记
  - 备忘录模板：无背景纯文字样式
  - 半撇模板：专业品牌风格，支持封面编辑

- 📝 **智能提取**
  - 自动从笔记内容中提取标题和摘要
  - 支持双击编辑标题和描述

- 🖼️ **封面生成**
  - 自动生成小红书风格封面
  - 支持横版 (16:9) 和竖版 (3:4) 比例
  - 半撇模板支持自定义背景图片

- 📥 **一键导出**
  - 导出单张图片或批量导出所有页面
  - 支持打包下载为 ZIP 文件

## 安装方法

### 方法一：BRAT 插件安装（推荐）

1. 在 Obsidian 中安装 BRAT 插件：
   - 打开设置 → 社区插件 → 浏览
   - 搜索 "BRAT" 并安装

2. 启用 BRAT 插件后：
   - 打开设置 → BRAT → Add Beta plugin
   - 输入仓库地址：`banpie/bp-ob-rednote`

3. 启用 "Banpie RedNote" 插件

### 方法二：手动安装

1. 克隆或下载本仓库
2. 将文件夹复制到你的 Obsidian vault 的 `.obsidian/plugins/` 目录下
3. 重启 Obsidian
4. 在设置中启用插件

## 使用方法

1. **创建笔记**：在你的 vault 中创建一篇笔记
2. **打开插件**：使用命令面板 (`Cmd/Ctrl + P`) 搜索 "RedNote" 或点击侧边栏图标
3. **编辑内容**：
   - 在左侧编辑区输入内容
   - 右侧实时预览效果
4. **切换模板**：点击模板选择器切换不同风格
5. **编辑标题/摘要**（半撇模板）：双击标题或摘要区域进行编辑
6. **导出图片**：点击导出按钮获取小红书图片

## 模板说明

| 模板 | 封面页 | 适用场景 |
|------|--------|----------|
| 默认模板 | ❌ 无 | 简洁笔记 |
| 备忘录模板 | ❌ 无 | 纯文字笔记 |
| 半撇模板 | ✅ 有 | 品牌内容 |
