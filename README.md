# Banpie RedNote (小红书图文卡片生成器)

这是一个基于 Note to RED 源码修改并增强的 Obsidian 插件，可以将 Obsidian 中的 Markdown 笔记一键转换为小红书风格的精美图文卡片。

## 主要功能

- 自动识别 Markdown 中的内容并生成排版精美的卡片。
- 特别针对半撇风格的内容进行了样式和排版的优化适配。
- 支持一键导出并保存为图片。

### 🎨 多模板支持
- **默认模板**：简洁风格，适合常规笔记（无封面页）
- **备忘录模板**：无背景纯文字样式（无封面页）
- **半撇模板**：专业品牌风格，支持封面编辑

### 📝 智能提取
- 自动从笔记内容中提取标题和摘要
- 半撇模板支持双击编辑标题和描述

### 🖼️ 封面生成
- 自动生成小红书风格封面
- 支持横版 (16:9) 和竖版 (3:4) 比例
- 半撇模板支持自定义背景图片

### 📥 一键导出
- 导出单张图片或批量导出所有页面
- 支持打包下载为 ZIP 文件

## 安装方法

### 方法一：BRAT 插件安装（推荐）

1. 在 Obsidian 中安装 BRAT 插件：
   - 打开设置 → 社区插件 → 浏览
   - 搜索 "BRAT" 并安装

2. 启用 BRAT 插件后：
   - 打开设置 → BRAT → Add Beta plugin
   - 输入仓库地址：`banpie/bp-ob-rednote`

3. 启用 "Banpie RedNote" 插件

### 方法二：手动安装

由于本修改版插件未上架 Obsidian 官方商店，你需要进行手动安装：

1. 点击本仓库绿色的 `<> Code` 按钮，选择 **Download ZIP** 将本仓库下载到电脑。
2. 解压下载的 ZIP 文件（通常文件夹会叫 `note-to-red-main`）。
3. 将解压出来的文件夹重命名为 `banpie-rednote` (非常重要，名字不能错)。
4. 找到你的 Obsidian 仓库所在位置，进入 `.obsidian/plugins` 目录。
5. 将重命名的 `banpie-rednote` 文件夹放入 `.obsidian/plugins` 目录中。
   > 最终路径应该是: `你的仓库目录/.obsidian/plugins/banpie-rednote/main.js` 等。
6. 重启 Obsidian。
7. 进入设置 `Settings` -> `第三方插件 (Community plugins)`，确保**关闭安全模式 (Safe mode)**。
8. 在列表中找到 **Banpie RedNote** 并开启，即可使用！

## 使用方法

1. **创建笔记**：在你的 vault 中创建一篇笔记
2. **打开插件**：使用命令面板 (`Cmd/Ctrl + P`) 搜索 "RedNote" 或点击侧边栏图标
3. **编辑内容**：
   - 在左侧编辑区输入内容
   - 右侧实时预览效果
4. **切换模板**：点击模板选择器切换不同风格
5. **编辑标题/摘要**（半撇模板）：双击标题或摘要区域进行编辑
6. **导出图片**：点击导出按钮获取小红书图片

## 模板说明

| 模板 | 封面页 | 适用场景 |
|------|--------|----------|

| 备忘录模板 | ❌ 无 | 纯文字笔记 |
| 半撇模板 | ✅ 有 | 品牌内容 |
